# C 练习实例7 - 素数

> 素数又叫质数<br>
百度百科介绍: 质数是指在大于1的自然数中，除了1和它本身以外不再有其他因数的自然数。 <br>
翻译成白话就是: 只能被 1 和 它本身整除的数为素数

***题目:*** 输出100以内的所有素数
***程序分析:*** 利用循环得到100以内的所有数, 利用取摸运算判断是否能被整除

## 代码
```c
for (i = 2; i < 100; i++) {
    for (j = 2; j < 100; j++) {
        if (i % j == 0) { /* 取余数 */
            break;
        }
    }
   if (j >= i) {
        printf("%d\n", i);
   }
}
```
