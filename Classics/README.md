<b>题目:</b> 有1、2、3、4个数字，能组成多少个互不相同且无重复数字的三位数？都是多少？ <br>
<b>程序分析:</b> 可填在百位、十位、个位的数字都是1、2、3、4。组成所有的排列后再去 掉不满足条件的排列. <br>
